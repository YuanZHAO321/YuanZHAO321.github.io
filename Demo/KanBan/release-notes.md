首个正式版本。一块和风纸感的复习看板,为长期记忆而生。macOS(Apple Silicon / Intel)与 Windows 同步发行。

## ✨ 亮点

- **四列复习流** — 未着手 → 復習中 → 要再読 → 修得,按掌握程度流转卡片
- **科目分类筛选** — 自定义科目与配色,顶部标签一键过滤
- **拖拽操作** — 列间移动、列内排序,触屏可用 ◀ ▶ 按钮
- **截止日提醒** — 「〆」记号,临期 3 天变朱红,逾期显示天数
- **印章语言** — 重点卡片盖「重」印,已掌握自动盖「済」朱印
- **数据本地保存** — 无账号、无联网,支持 JSON 导出 / 导入备份
- **日式设计** — 明朝体、汉数字日期、暖帘色带、円相背景

## 📦 下载与安装

| 平台 | 文件 | 大小 |
|---|---|---|
| macOS · Apple Silicon | `復習看板-1.0.0-arm64.dmg` | 114 MB |
| macOS · Intel | `復習看板-1.0.0.dmg` | 116 MB |
| Windows 10/11 · x64 | `復習看板-Setup-1.0.0.exe` | 97 MB |

**macOS**:打开 DMG,把 **復習看板.app** 拖入「应用程序」。应用未签名,首次启动请 **右键 → 打开** 绕过 Gatekeeper(仅需一次)。

**Windows**:运行安装程序,可自选安装目录。SmartScreen 弹窗时点 **「更多信息」→「仍要运行」**。

不想装 App?仓库里的 `revision-kanban.html` 用浏览器打开即可,功能完全一致。
